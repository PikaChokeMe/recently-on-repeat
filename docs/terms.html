What Listen on Repeat is

Listen on Repeat is a source-available Discord profile widget updater. It is designed to fetch Spotify track metadata and push that data into a custom Discord profile widget.

The current version is a self-hosted/template project. Users are expected to configure their own Discord Developer Application, Spotify Developer Application, GitHub Actions workflow, and secrets.
