Discord profile widget

Listen on Repeat

Listen on Repeat is a source-available Discord profile widget updater that pulls Spotify track data and pushes it into a custom Discord widget.

I wrote this for personal use: to show the top 5 current Spotify earworms from my account on my Discord profile.

This project is shared so people can recreate a similar setup with their own Discord app, Spotify app, and GitHub Actions workflow without reinventing the whole wheel.

How it works

The widget itself does not directly call Spotify. A scheduled updater fetches Spotify data, formats it for the Discord widget fields, and pushes the latest snapshot to Discord.

Secrets like Spotify refresh tokens, Discord bot tokens, and client secrets should stay in a trusted environment such as GitHub Actions secrets. They should never be exposed in the widget, browser JavaScript, or public repository files.

Self-hosted template

This is currently a self-hosted/template project. People using it for themselves are expected to create and configure their own Discord Developer Application, Spotify Developer Application, and GitHub Actions secrets.
